# heroes-app

## Description

This is an educational project for learning Angular. It is an application for managing heroes.

## Installation

Clone the repository:

```
git clone vakulio/HeroesApp
```

Navigate to the project directory:

```
cd heroes-app
```

Install the dependencies:

```
npm install
```

## Usage

Start the development server:

```
npm start
```

Build the project:

```
npm run build
```

Watch for changes and rebuild in development mode:

```
npm run watch
```

Run tests:

```
npm test
```

## Scripts

- `ng`: Angular CLI command.
- `start`: Start the development server.
- `build`: Build the project.
- `watch`: Watch for changes and rebuild in development mode.
- `test`: Run tests.
